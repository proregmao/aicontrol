## 使用说明

### 代码格式
- 所有代码均为16进制格式，前缀0x
- 代码范围：0x004F - 0x00D8
- 每个品牌可能有多个代码版本（老款/新款）

### 使用方法
1. **查找品牌代码**：根据空调品牌查找对应的16进制代码
2. **生成控制指令**：将代码嵌入到CX-IR002E的JSON指令中
3. **测试验证**：发送指令测试空调响应

### 指令示例
```json
{
  "irout0h": "01060010[品牌代码]49CF",
  "res": "123"
}
```

例如控制格力空调（代码0x006E）：
```json
{
  "irout0h": "01060010006E49CF",
  "res": "123"
}
```

## 注意事项

1. **品牌兼容性**：同一品牌可能有多个代码，需要逐一测试
2. **型号差异**：不同型号的空调可能需要不同的代码
3. **学习模式**：如果预设代码无效，可使用红外学习功能
4. **测试建议**：建议先测试开关机功能，再测试其他功能
